# テストデータ作成方法

## 実行方法

### 方法1: npmスクリプトを使用（推奨）

```bash
npm run seed
```

### 方法2: バッチファイルを使用

`scripts/seed.bat` をダブルクリック

### 方法3: 直接実行

```bash
node prisma/seed.js
```
