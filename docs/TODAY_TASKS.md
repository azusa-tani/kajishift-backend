### 2. express-rate-limitの警告解消（バックエンド）

#### 症状
```
ValidationError: The Express 'trust proxy' setting is true, which allows anyone to trivially bypass IP-based rate limiting.
```

#### 対処方法
- [ ] `src/index.js`の`trust proxy`設定を修正
  - 現在: `app.set('trust proxy', true);`
  - 変更後: `app.set('trust proxy', 1);`（最初のプロキシのみを信頼）

**参考**: `docs/DEPLOYMENT_HANDOVER.md`の「express-rate-limitの警告」セクション参照
